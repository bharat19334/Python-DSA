# 🚀 DSA Pattern Wise Questions

A comprehensive collection of Data Structures and Algorithms problems organized pattern-wise to strengthen problem-solving skills and prepare for coding interviews.

---

## 📌 About Repository

This repository contains DSA questions categorized according to different problem-solving patterns. The objective is to master each pattern individually and build a strong foundation for technical interviews, competitive programming, and software engineering roles.

---

## 🎯 Learning Goals

* Master Data Structures & Algorithms
* Improve Problem-Solving Skills
* Prepare for Coding Interviews
* Build Consistency in DSA Practice
* Learn Multiple Approaches to Solve Problems
* Track Progress Pattern-Wise

---

## 📂 DSA Patterns Covered

### 🔹 Arrays

Questions related to array traversal, manipulation, prefix sums, and optimization.

### 🔹 Strings

Problems involving string processing, manipulation, and pattern matching.

### 🔹 Hashing

Questions solved using HashMap, HashSet, Frequency Count, and Lookup Techniques.

### 🔹 Sorting Algorithms

* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Heap Sort
* Counting Sort
* Radix Sort

### 🔹 Two Pointers

Problems involving multiple pointers moving through data structures.

### 🔹 Sliding Window

Fixed Window and Variable Window optimization problems.

### 🔹 Binary Search

Classic and advanced Binary Search based problems.

### 🔹 Stack

Questions involving monotonic stacks, expression evaluation, and stack operations.

### 🔹 Queue & Deque

Queue implementation and optimization problems.

### 🔹 Linked List

Singly, Doubly, Circular Linked Lists and related interview questions.

### 🔹 Trees

Binary Trees and Tree Traversal based problems.

### 🔹 Binary Search Trees (BST)

Insertion, Deletion, Search, and BST-specific questions.

### 🔹 Heap / Priority Queue

Min Heap and Max Heap based optimization problems.

### 🔹 Recursion

Recursive thinking and recursive problem-solving techniques.

### 🔹 Backtracking

Decision tree exploration and constraint satisfaction problems.

### 🔹 Greedy Algorithms

Optimization problems solved through local optimal choices.

### 🔹 Dynamic Programming

Memoization and Tabulation based problems.

### 🔹 Graphs

BFS, DFS, Topological Sort, Shortest Path, MST, and more.

### 🔹 Trie

Prefix Tree based problems.

### 🔹 Bit Manipulation

Binary operations and bitwise optimization problems.
